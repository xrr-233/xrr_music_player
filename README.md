# 画大饼

本项目将使用Vue+Electron+Flask混合框架开发

# 前置操作

## node.js与npm安装

欲运行此程序，首先保证你已在计算机上安装node.js

http://nodejs.cn/

另外最好将npm升至最新版本

```
npm install -g npm
```

安装完成后在cmd中测试nodejs和npm是否安装成功

```
node -v
npm -v
```

## npm依赖包安装

```
npm install
cd xrr_music_player
npm install
```

理论上会有vulnerabilities，忽略~~（人菜）~~即可

# 运行

```
npm run electron:serve
```
